# Hotel Recommendation System

## Description
This project is a Machine Learning based Hotel Recommendation System developed using Python and Flask.

## Features
- User Login
- Hotel Recommendations using TF-IDF
- Booking System
- Thank You Page

## Technologies Used
- Python
- Flask
- Scikit-learn
- HTML, CSS

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
